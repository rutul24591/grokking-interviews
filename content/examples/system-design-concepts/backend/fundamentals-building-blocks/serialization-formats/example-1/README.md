# Serialization Formats Example

This example compares JSON, MessagePack (simulated), and a binary buffer format for payload size and parsing cost.

## Files
- `formats.js`: Serializer/deserializer implementations.
- `benchmark.js`: Compares payload sizes.
- `client.js`: Runs the comparison.

## Run
`node content/examples/backend/fundamentals-building-blocks/serialization-formats/example-1/client.js`
