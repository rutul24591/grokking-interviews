# Horizontal vs Vertical Scaling Example

This example simulates scaling decisions for a service with CPU, memory, and concurrency constraints. It compares scale-up vs scale-out trade-offs.

## Files
- `capacity-model.js`: Simple capacity calculator.
- `autoscaler.js`: Decides whether to scale up or out.
- `client.js`: Runs scenarios.

## Run
`node content/examples/backend/fundamentals-building-blocks/horizontal-vs-vertical-scaling/example-1/client.js`
