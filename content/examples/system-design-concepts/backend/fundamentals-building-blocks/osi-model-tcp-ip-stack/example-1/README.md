# OSI Model & TCP/IP Stack Example

This example traces a request through conceptual layers using small adapters to illustrate encapsulation and data flow.

## Files
- `layers.js`: Layer pipeline definitions.
- `packet.js`: Encapsulation helpers.
- `client.js`: Sends a request through layers.

## Run
`node content/examples/backend/fundamentals-building-blocks/osi-model-tcp-ip-stack/example-1/client.js`
