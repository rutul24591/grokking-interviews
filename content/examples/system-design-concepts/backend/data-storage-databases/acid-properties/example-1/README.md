# ACID Properties Example

This example simulates transactions with atomicity, consistency checks, isolation via locks, and a durability log.

## Files
- `ledger.js`: In-memory account ledger.
- `transaction-manager.js`: Transaction lifecycle and WAL.
- `demo.js`: Runs transfer scenarios.

## Run
`node content/examples/backend/data-storage-databases/acid-properties/example-1/demo.js`
