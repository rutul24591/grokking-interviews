# Example 1 — Semantic landmarks + document structure (end-to-end)

Demonstrates:

- semantic landmarks: `<header>`, `<nav>`, `<main>`, `<footer>`
- clear content boundaries: `<article>`, `<section>`, `<aside>`
- a visible “Skip to content” link and focusable main heading
- breadcrumbs as navigation (`<nav aria-label="Breadcrumb">`)
