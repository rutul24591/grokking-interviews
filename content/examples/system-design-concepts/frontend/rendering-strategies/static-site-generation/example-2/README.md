# Example 2 — SSG Pattern: Build-Time Search Index

## What it shows
- A Node.js build step that creates `content/search-index.json`.
- A client-side search UI that uses the static index (no runtime server).

## Run it (after copy-paste)
```bash
pnpm install
pnpm dev
```

## Build it
```bash
pnpm build
pnpm start
```
