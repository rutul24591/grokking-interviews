# Example 2 — CSR Performance: Virtualization + Deferred Input

## What it shows
- Naive list rendering vs virtualized list rendering (10k items).
- `useDeferredValue` to keep inputs responsive under expensive renders.

## Run it (after copy-paste)
```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000` and toggle **Naive / Virtualized**.
