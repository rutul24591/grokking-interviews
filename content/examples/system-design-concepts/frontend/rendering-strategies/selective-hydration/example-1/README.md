# Example 1 — Selective Hydration: Suspense + Event Replay

## Run it (after copy-paste)
```bash
pnpm install
pnpm dev
```

Open:
- `http://localhost:3000`

### Try event replay
1. Hard refresh the page.
2. Immediately click **Queued button** a few times.
3. After ~3s, hydration finishes and the clicks should be applied.
