This example focuses on IP parsing, CIDR math, and subnet policy checks.
`ip-utils.js` normalizes IPv4/IPv6 and converts to integers for comparison.
CIDR containment is how firewalls and allowlists decide if an IP is permitted.
`subnet-policy.js` models allow/deny rules similar to security groups.
The client shows private-range detection and mixed IPv4/IPv6 handling.
