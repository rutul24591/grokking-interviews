# Networking Fundamentals Example

This example simulates routing, NAT, and firewall rules to show how packets move across a network boundary.

## Files
- `network.js`: Simple router and NAT simulation.
- `firewall.js`: Allow/deny rules.
- `client.js`: Sends packets through the network.

## Run
`node content/examples/backend/fundamentals-building-blocks/networking-fundamentals/example-1/client.js`
