# SEO & discoverability as an NFR

SEO isn’t “add meta tags”; it’s a system property:
- stable URLs,
- consistent canonicalization,
- crawlability (robots/sitemap),
- and structured data where relevant.

This example shows a minimal but production-shaped setup for content pages.

It also includes `GET /api/seo/preview?slug=...` to demonstrate server-side canonical/metadata generation (useful for link preview services, debuggers, and audits).
